## Recording Tips

1. **Browser setup**: Maximize Chrome window, hide bookmarks bar, use a clean profile
2. **Mouse movement**: Slow, deliberate — the mouse IS the narrator
3. **Pause on key moments**: Hold for 2-3 seconds on each metric card, AI analysis paragraph, and the advisor decision panel
4. **Text overlay timing**: Each overlay should stay on screen for at least 3 seconds
5. **Transitions**: Simple fade between sections, no flashy effects
6. **Re-record**: Plan for 3+ takes. The third take is usually the keeper
7. **Final check**: Watch at 1x speed with fresh eyes before submitting
